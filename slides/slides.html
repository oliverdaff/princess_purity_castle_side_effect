Goomba problem

Defeating a Goomba

Stomp it and it turns into a coin


trait Stompable[T] {
  def stomp():T
}
case class Coin()
case class Goomba() extends Stompable[Coin] {
  def stomp() = Coin()
}

def stomp[T](s:Stompable[T]) = s.stomp()

The function stomp is our pipe, that transforms from a Stompable to a Coin.

Mario has a side kick called Yoshi who can eat stompables


class Yoshi {

  def eat[T](s:Stompable[T]) = stomp(s)

}
          

val yoshi = new Yoshi()

val goomba = new Goomba()

yoshi.eat(goomba)

What happens when Yoshi misses?


val yoshi = new Yoshi()

yoshi.eat(null)
